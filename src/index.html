REST Horoscope App

A REST API and web application built using: Python and the Flask framework for the API, and JavaScript & React along with HTML/CSS and Bootstrap 5. The Python REST API returns a JSON object with the appropriate horoscope data. The React Web Application takes the user's horoscope sign and day, uses it to send a GET request to the correct API endpoint, and parses the JSON object for the necessary horoscope data. Finally, the user's horoscope is displayed to the user asynchronously so that the entire page doesn't have to be re-rendered each time a user selects a different sign and day.
