Python Stock App

This project was made using Python, Plotly and Dash, and the yfinance api. My goal was to make a simple dashboard application that would output a candlestick chart, standard moving averages, and some commonly used indicators below the main plots. The legend is interactive and displays the 5, 15, 50, and 200 day moving averages along with support and resistance levels when the user clicks on them. The algorithm used is from a medium article, found within the References and Sources section below, that I applied to my own project to help with visualizing the chosen stock's behavior. The application itself is hosted on Microsoft Azure. Click the button below to see it in action!

Features and Highlights

  • Candlestick chart showing open, high, low, close values
  • 5, 15, 50, 200 standard moving averages
  • Volume, RSI, and MACD indicators
  • Support/Resistance value detection and plotting
  • Interactive legend

Motivations for project

During the COVID-19 pandemic, we all saw and experienced things that were outside of our daily norm. Toilet paper was flying off shelves, conducting business and socializing was done virtually, and the stock market and economy seemed to be heading down towards the abyss. Then, thanks to the economic stimulus and near-zero interest rates, the stock market made a comeback that attracted an unprecedented amount of volume and volatility.


To better visualize and analyze the trends and price movements of a particular stock, I wanted to make an application that would equip users with the indicators and moving averages they'd need to implement some basic trading strategies. While trying to make sense of the market myself as a beginner, I found the UI and settings of some applications to be cumbersome or confusing, so I wanted to offer a more streamlined approach.

Why I Built This Project

I built this project because I wanted to make an application that consumed data from an API, which is the yfinance API in this case. During one of my programming classes, I was introduced to the pandas library and learned how to plot and analyze data using the dataframes object and the Matplotlib plotting library. Since I already had some experience with using dataframe objects to output a graph, I wanted to build on top of that knowledge. After google searching around for different ways of analyzing stock data, I came across the Plotly and Dash libraries and decided that I wanted to learn more about them.

What I Learned

While developing this stock application, I learned some really interesting things. I learned how to consume data from the yfinance API that pulls its data from the Yahoo! Finance website. I also got experience with using the built-in functions to plot the interactive candlestick chart using the Plotly graphing library. From there, I learned how to plot multiple subplots on a single page instead of relying on just the single candlestick chart. I chose to make a MACD(Moving Average Cross Divergence) subplot, a volume subplot, and also an RSI(Relative Strength Index) subplot. I came across a few different ways of calculating the RSI and MACD values, and decided to use the Technical Analysis library to derive the values I needed. Using the Dash framework, I learned how to use a callback function to change the value of the stock choice used to output the graphs, indicators, and the support and resistance values. Finally, I deployed the working application using Microsoft Azure and Github for version control.
